# 文章发布前检查清单

## 读者和主题

- [ ] 目标读者明确。
- [ ] 核心问题具体。
- [ ] 标题和正文内容一致。
- [ ] 没有为了吸引点击而夸大承诺。

## 结构

- [ ] 开头能快速进入问题。
- [ ] 每个小节只解决一个主要问题。
- [ ] 小节顺序符合读者理解路径。
- [ ] 结尾有总结或行动建议。

## 内容

- [ ] 有具体例子，不只是抽象观点。
- [ ] 数据、引用、事实已核实。
- [ ] 专业判断和个人观点有区分。
- [ ] 没有明显逻辑跳跃。

## 表达

- [ ] 删除口号式表达。
- [ ] 删除重复段落。
- [ ] 长句已拆短。
- [ ] 术语已解释或替换成读者能理解的说法。

## 发布

- [ ] 标题已确认。
- [ ] 封面图已确认。
- [ ] 摘要已确认。
- [ ] 排版已预览。
- [ ] 末尾互动问题已确认。
