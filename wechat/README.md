# 微信公众号写作工作区

这个目录用于管理公众号文章从选题到发布复盘的完整流程。

## 推荐使用顺序

1. 在 [`topic-pool.md`](topic-pool.md) 记录选题。
2. 在 [`material-library.md`](material-library.md) 收集素材。
3. 用 [`templates/article-brief.md`](templates/article-brief.md) 明确单篇文章任务。
4. 用 [`templates/article-outline.md`](templates/article-outline.md) 生成文章大纲。
5. 基于 [`templates/article-template.md`](templates/article-template.md) 创建草稿，放入 `drafts/`。
6. 用 [`templates/article-review-checklist.md`](templates/article-review-checklist.md) 审稿。
7. 发布后把最终稿放入 `published/`。
8. 在 [`reviews/monthly-review.md`](reviews/monthly-review.md) 做月度复盘。

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `public-account-workflow.md` | 总体工作流说明 |
| `topic-pool.md` | 长期选题池 |
| `material-library.md` | 素材、案例、观点、数据来源 |
| `publishing-calendar.md` | 发布排期 |
| `title-bank.md` | 标题库和标题测试 |
| `templates/article-brief.md` | 单篇文章写作任务书 |
| `templates/article-outline.md` | 大纲模板 |
| `templates/article-template.md` | 正文草稿模板 |
| `templates/article-review-checklist.md` | 发布前检查清单 |
| `drafts/` | 未发布草稿 |
| `published/` | 已发布文章 |
| `reviews/` | 周期复盘 |

## 当前最小目标

连续 4 周，每周完成 1 篇文章：

```text
第 1 周：跑通流程，不追求完美。
第 2 周：优化选题和开头。
第 3 周：优化标题和结构。
第 4 周：基于数据做复盘，确定长期栏目。
```
